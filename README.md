# PartyHall

Welcome to the PartyHall docs. You will find there the instructions to build your own PartyHall booth and customize it.

## Required tools

All of these tools are optional but you will need to think about how to do something equivalent if you don't have some of them.

- 3D Printer (Required for the buzzers)
- Handsaw (Or a shop that can cut wood for you)
- Hot glue gun + hot glue sticks
- A power drill
- Wood paint + brush (I've made mine fully black but you can use the color you want)
- Wood varnish
